# 2019workshop

## 如要安裝項目依賴
```
npm install
```

### 使用線上熱修復（web)
```
npm run serve
```

### 產出編譯web，請放至雲端硬碟dsa-vue資料夾裡，並加上當天日期，如遇重複請加上流水號，以區別之
```
npm run build
```

### web語法測試
```
npm run test
```

### 修正相依性
```
npm run lint
```
### 編譯當前平台版本應用程式，取setup.exe放置dsa-vue
```
npm run electron:build
```
